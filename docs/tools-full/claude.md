# ✍️ Claude 使用指南

## 简介
Claude 是 Anthropic 公司推出的对话模型，擅长生成自然语言，适合处理长文本、总结、脚本和编辑任务。

## 推荐用途
- 长文写作（公众号、视频号文案）
- 内容总结（长文章 → 精华）
- 对话式脚本生成

## 上手方式
- 官网地址：[https://claude.ai](https://claude.ai)
- 免费版本已支持 Claude 3（性能接近 GPT-4）

## 推荐 Prompt 示例
```
请你根据以下提纲写一篇 300 字左右的视频解说文案，语气理性但真诚，适合中年男性受众...
```
