## 个人学习路线（示例）

- **阶段 1：工具熟练度**
  - 目标：把常用工具的“边界”跑清楚
  - 方式：每周一个工具深测 + 输出评测

- **阶段 2：工作流构建**
  - 目标：从“点工具”到“组合拳”
  - 方式：围绕内容创作、IP 建设沉淀 SOP

- **阶段 3：可复制的产品化能力**
  - 目标：让别人也能跟着你的方法跑
  - 方式：把 SOP + Prompt + Demo 打包，变成一份“方法论 + 资源包”
